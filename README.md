# gestor_academico
"A simple project for praticing Spring Boot + Spring Data JPA"

Aqui está um gestor acadêmico, uma versão melhorada do gestor de alunos onde o objetivo foi praticar a persistência de dados com Spring JPA, a lógica de negócio com Service e uma estilização front-end para treinar um pouco.

-----------
"Here is an academic manager, an improved version of the student manager where the goal was to practice data persistence with Spring JPA, business logic with a Service, and some front-end styling for practice."
